Button按钮

标记了一个（或封装一组）操作命令，响应用户点击行为，触发相应的业务逻辑。

代码演示

按钮类型

按钮有五种类型：默认按钮、首选项按钮、成功按钮、警告按钮、危险按钮。

通过设置 class 为btn-default btn-primary btn-success btn-warning btn-danger 可分别创建按钮。

												<button class="btn btn-default">默认按钮</button>
											
												<button class="btn btn-primary">首选项按钮</button>
											
												<button class="btn btn-success">成功按钮</button>
											
												<button class="btn btn-warning">警告按钮</button>
											
												<button class="btn btn-danger">危险按钮</button>

按钮尺寸

按钮有大、默认、小三种尺寸。

通过设置 class 为 btn-lg btn-sm 分别把按钮设为大、小尺寸。若不设置，则尺寸为默认。

												<button class="btn btn-primary btn-lg">大按钮</button>
											
												<button class="btn btn-primary">默认按钮</button>
											
												<button class="btn btn-primary btn-sm">小按钮</button>

禁用状态

当按钮处于不可用状态时,只需要将属性 class值中增加 disabled

												<button class="btn btn-default disabled">默认按钮</button>
											
												<button class="btn btn-primary disabled">首选项按钮</button>
