徽章

一般出现在通知图标或头像的右上角，用于显示需要处理的消息条数，通过醒目视觉形式吸引用户处理。

代码演示

数字徽标

username

.badge标签添加自定义属性 count 值为number 

<a href="#" class="badge" count="234"></a>

有限制的数字徽标

.badge标签添加自定义属性 limited 值为number

<a href="#" class="badge" count="123" limited="99"></a>

不同颜色的数字徽标

.badge默认
.badge添加 .success显示绿色背景
.badge添加 .error显示红色背景 

<a href="#" class="badge" count="234" limited="99"></a>
	
<a href="#" class="badge success" count="234" limited="99"></a>
	
<a href="#" class="badge error" count="234" limited="99"></a>
